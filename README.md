Introduction to shangxian.app

## Service

- Docker - 容器化
- Jenkins - 构建、编译工具
- Nginx - Web Server
- Node.js - 应用
- Kibana - 日志可视化
- Shadowsocks - v.p.n
- MTProxy - Telegram Proxy

## Env

### common

- `$LOCAL_DIR` - 软件安装目录
- `$NGINX_DIR` - Nginx 安装目录，如使用 `$NGINX_DIR/bin/nginx` 为可执行文件
- `$NGINX_CONF_PATH` - Nginx 配置文件路径，注意：这是 `nginx.conf` 文件路径，不是目录
- `$NGINX_VHOST_DIR` - Nginx 虚拟主机目录
